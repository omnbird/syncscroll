# SyncScroll 演示文档

## 为什么要开发这个插件

我经常在 Obsidian 中阅读英文技术文档，并同时进行翻译整理成中文笔记。但在翻译过程中，经常会遇到这样的问题：

- 有些专业术语记不准确，需要反复回看原文确认
- 翻译完后发现表述不够准确，需要对照原文重新修改
- 两份文档内容一长一短，滚动后很难找到对应位置

以前我只能在两个文件之间来回切换，或者手动调整滚动位置来对比查看，非常麻烦。于是我开发了这个 SyncScroll 插件——只要三连击当前正在翻译的某一行，另一边就能自动跳转到对应位置并保持水平对齐，大大提升了翻译和对比阅读的效率。
